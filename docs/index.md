# allinkl Provider

Manages all-inkl.com (KAS) hosting resources through the KAS SOAP API: DNS
records, mail accounts, mail forwards and subdomains, with the account's
domains as a data source.

~> **Unofficial.** Not affiliated with, endorsed by, or supported by
all-inkl.com (Neue Medien Münnich GmbH). "all-inkl" and "KAS" name the service
this talks to, nothing more.

## Example Usage

```terraform
provider "allinkl" {
  # or set KAS_LOGIN / KAS_PASSWORD environment variables
  login    = "w0123456"
  password = var.kas_password
}
```

The KAS API enforces flood protection between requests; the provider serializes
calls and honors the announced delays, so large plans apply slowly by design.
